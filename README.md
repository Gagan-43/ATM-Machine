# 🏧 ATM Machine Simulation

A responsive and interactive ATM Machine Simulation built using **HTML, CSS, and JavaScript**. This project replicates the core functionalities of an Automated Teller Machine (ATM), allowing users to securely log in with a PIN, check their account balance, deposit and withdraw money, and view transaction history. It is designed to demonstrate JavaScript concepts such as DOM manipulation, event handling, form validation, and dynamic UI updates.

---

## 🚀 Features

- 💰 Balance Inquiry
- ➕ Deposit Money
- 💸 Withdraw Cash
- 📜 Transaction History
- ⚠️ Insufficient Balance Validation
- 📱 Responsive Design
- 🎨 Clean and User-Friendly Interface

## 📖 How It Works

1. Enter the correct ATM PIN.
2. Access the ATM dashboard.
3. Choose one of the available operations:
   - Check Balance
   - Deposit Money
   - Withdraw Money
   - View Transaction History
4. The balance updates automatically after each successful transaction.
5. Error messages are displayed for invalid PINs or insufficient balance.
